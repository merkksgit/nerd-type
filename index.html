Introduction


NerdType is a minimalistic typing game developed using JavaScript, HTML, and CSS. The game challenges the user to type the correct words within a time limit, with the possibility of gaining extra time for each correct word.

Features


Players must type a randomly chosen word before time runs out. Each correct word adds extra time. The game picks a random word from a predefined word list in each round. The total time spent in the game is displayed. Built with Bootstrap to ensure a smooth experience across different screen sizes.

How to play


Choose a game type from the navigation bar. Click the Start button to begin the game. Type the word displayed in the input field before the time runs out. For every correct word, additional time is added to the countdown. The game ends when time runs out, and the page automatically reloads to restart.
